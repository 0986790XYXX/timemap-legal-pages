# TimeMap Legal Pages

Public legal documents for the **TimeMap (时间可视化)** iOS app, hosted via GitHub Pages.

The TimeMap source code lives in a separate private repository. This repository **only** contains:
- Terms of Service (Chinese + English)
- Privacy Policy (Chinese + English)

## How to update

1. Edit any `docs/*.md` file
2. `git commit && git push`
3. GitHub Pages auto-rebuilds in ~1 minute
