# 隐私政策

**应用名称**：时间可视化（TimeMap）
**开发者**：[开发者姓名 / Developer Name]
**生效日期**：2026-05-23
**版本**：v1.0

本隐私政策说明「时间可视化（TimeMap）」（以下简称"本应用"）如何收集、使用、存储和保护您的个人信息。请您在使用本应用前仔细阅读并理解本政策的全部内容。

---

## 1. 我们收集的信息

### 1.1 您主动提供的信息

- **账号注册信息**：电子邮箱、密码（密码以加密哈希形式存储，开发者无法看到您的原始密码）
- **任务记录**：您在本应用中创建的任务标题、分类、时长、日期等内容

### 1.2 自动收集的信息

- **设备与崩溃信息**：设备型号、操作系统版本、应用版本，以及应用崩溃时的诊断信息。该信息用于排查问题、改进应用稳定性。
- **使用分析信息**：通过第三方分析服务收集的匿名使用统计（例如打开次数、停留时长、功能使用频率），不包含可识别您身份的信息。

### 1.3 我们不收集的信息

- 我们**不**收集您的姓名、电话号码、身份证号、地理位置（GPS / 精确定位）
- 我们**不**读取您的通讯录、相册、相机、麦克风
- 我们**不**收集第三方账号信息（本应用不支持第三方登录）

## 2. 我们如何使用信息

我们仅在以下场景使用您的信息：

- 提供应用核心功能（记录、同步、统计、导出）
- 验证您的账号身份
- 修复故障、改进应用质量
- 在法律法规要求时配合执法部门调查

我们**不会**将您的个人信息用于广告投放，**不会**将您的任务记录出售或出租给任何第三方。

## 3. 信息存储与第三方服务

### 3.1 存储位置

- 您的账号和任务记录存储在 **Supabase**（https://supabase.com）提供的云数据库中。Supabase 是一家位于美国的数据库服务商，您的数据可能存储在其全球数据中心。
- Supabase 的隐私政策：https://supabase.com/privacy

### 3.2 涉及的第三方服务

| 服务 | 用途 | 隐私政策 |
|------|------|----------|
| Supabase | 账号鉴权、任务数据存储 | https://supabase.com/privacy |
| Expo / EAS | 应用构建、推送通知（如有） | https://expo.dev/privacy |
| [第三方分析服务名 / Third-party analytics] | 匿名使用统计 | [填写隐私政策链接] |

### 3.3 数据保留

您账号下的任务记录将一直保留，直到您主动删除或注销账号。注销后，您的个人数据将在 30 天内被永久删除。

## 4. 信息安全

- 所有客户端与服务器之间的通讯均通过 HTTPS / TLS 加密传输。
- 密码以加密哈希形式存储，开发者无法获取明文密码。
- 我们采取合理的技术和管理措施防止信息泄漏、损坏或丢失，但请理解互联网环境并非绝对安全，我们无法对您的信息安全作出绝对的承诺。

## 5. 您的权利

您对自己的信息享有以下权利：

- **访问与导出**：您可以随时通过应用内的"导出"功能，以 Excel / JSON / CSV 格式获取您的全部任务数据。
- **更正与删除**：您可以在应用内编辑或删除任意一条任务记录。
- **注销账号**：如需注销账号、删除全部数据，请联系开发者邮箱。我们将在 7 个工作日内处理，并在 30 天内完成数据删除。
- **撤回同意**：您可以选择停止使用本应用。

## 6. 儿童隐私

本应用面向 17 岁及以上用户。我们不会有意收集 13 岁以下儿童的个人信息。如果您是儿童的监护人，发现孩子未经您的同意向我们提供了信息，请联系我们删除。

## 7. 政策修改

我们可能不时更新本隐私政策。重大更新将在本页面公布，并通过应用内通知告知您。您在政策修改后继续使用本应用，即视为接受修改后的政策。
